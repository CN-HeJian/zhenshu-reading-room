# 枕书：植物札记视觉升级设计

## 方向

采用“植物札记”方向：鼠尾草绿、森林墨色、米白纸张和珊瑚色小点缀。整体从旧版的宣纸平铺改为更轻盈的阅读日志，保留中文 serif 标题，同时用系统无衬线字体承载导航、统计和辅助信息。

## 页面结构

- 顶部导航变成半透明毛玻璃栏，品牌标记使用圆形森林绿印章。
- 首页 Hero 使用左右分栏，右侧以 CSS 植物/纸团装饰表达“思想生长”。
- 书架卡片采用圆角纸卡、柔和阴影和更明显的进度条；保留现有真实封面、链接和 6 本/页分页。
- 阅读足迹改为森林绿高对比度统计区，批注与日历使用浅色纸卡和鼠尾草绿交互状态。
- 移动端继续使用两列书架、一列阅读足迹和可滚动的日历布局。

## 保持不变

不改变数据格式、微信读书导出逻辑、每日 23:30 定时同步、搜索、分页、批注日历和阅读统计行为。只调整视觉样式与少量首页文案。

## 验收

- 桌面和移动端颜色、圆角、间距和焦点状态统一。
- 真实数据仍能加载，书架和批注交互不回归。
- 运行测试、lint、build，并在本地页面截图检查视觉层级。
